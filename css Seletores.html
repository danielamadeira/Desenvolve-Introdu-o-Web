css (Cascading Style Sheets)

CSS (Cascading Style Sheets) é uma linguagem de estilo que separa a apresentação visual (layout, cores, fontes) da estrutura de um documento web (HTML). Essa separação permite estilizar páginas de forma consistente, aplicar mudanças rápidas em todo o site e criar designs que se adaptam a diferentes dispositivos, tornando a experiência do usuário mais dinâmica e a manutenção do código mais fácil. A "cascata" no nome refere-se à hierarquia que determina qual estilo tem precedência em caso de conflitos. Para mais informações, Clique aqui!

O que é CSS? Linguagem de estilo: Define a aparência de documentos web, como cores, fontes, espaçamento e layout. Separação de conteúdo e estilo: Enquanto o HTML estrutura o conteúdo, o CSS cuida da apresentação visual, o que facilita a manutenção e o redesenho de sites. Aplicações: Usado para estilizar elementos em uma página web, seja na tela, no papel ou em outros formatos.

Como funciona Cascata: Define a ordem de prioridade das regras de estilo. A hierarquia é influenciada por três fatores: a origem do estilo (navegador, usuário ou autor), a especificidade do seletor (ID tem mais prioridade que classe, por exemplo) e a ordem em que as regras são definidas no código. Herança: Elementos filhos herdam estilos de seus elementos pais, facilitando a aplicação de regras comuns. Especificidade: Quanto mais específico o seletor (por exemplo, um seletor que usa um ID), maior a sua prioridade em relação a seletores mais gerais. Responsividade: O CSS é crucial para a responsividade, permitindo que o layout se ajuste a diferentes tamanhos de tela (como em celulares e TVs) usando recursos como as media queries.
